## 文件职责

本文件只保存每个考点的最新状态，是当前进度表。

- 只保留每个考点一行。
- 每次练习、批改、通关后更新本表。
- 不在本表追加历史快照。
- 通关历史写入 `09_考点通关记录.md`。

## 掌握状态

- 未学习：尚未进入该考点。
- 学习中：已开始讲解或练习。
- 半掌握：部分答对，但未达到通关条件。
- 需复习：答错或关键步骤不稳。
- 已掌握：达到当前训练强度的通关条件。

## 更新规则

1. 每次练习后更新已做题数。
2. 答对更新连续答对数。
3. 答错将连续答对数清零，并增加错题数。
4. 达到训练强度要求后标记为已掌握。
5. 如果错题本中该考点出现到期错题，下一步建议应优先提示错题回顾。

## 下一步建议示例

| 场景 | 建议写法 |
|---|---|
| 未学习 | 从讲解开始 |
| 刚进入考点 | 完成当前随堂练习 |
| 答对但未通关 | 继续做同类变式题 |
| 答错 | 先补讲薄弱点，再做基础变式题 |
| 部分正确 | 补齐关键词后再练 1 题 |
| 有到期错题 | 先回顾到期错题，再继续新题 |
| 连续错 2 次 | 提高当前优先级，插入补弱回合 |
| 已掌握 | 等待四选项选择 |
| 用户暂时跳过 | 标记需复习，后续回到本考点 |
