# FinaleForge Coach

FinaleForge Coach 是一个面向 Codex 的交互式期末复习教练 Skill。它会把当前
文件夹中的课程资料整理成可执行的复习路线，并通过持续的讲解、练习、批改、
错题复习和掌握度判断，帮助使用者把“看过资料”转化为“能够在考试中作答”。

它适用于数学、计算机、语言、法律、医学基础课等不同学科，不依赖固定课程结构。

## 它能做什么

- 扫描教材、讲义、PPT、PDF、Word 笔记、考试范围和老师划重点文件
- 结合题库、真题、样卷和复习提纲判断考点优先级
- 建立资料盘点、重点清单、考点覆盖矩阵和交互学习路线
- 让用户确认资料范围和学习路线，避免遗漏或错误理解考试范围
- 提供轻量、标准和强化等训练强度
- 每轮只讲一个小考点，控制信息量并保持真正的互动
- 讲解后出题并等待用户作答，不提前泄露答案
- 批改答案、分析错误原因、补充讲解并安排针对性重练
- 维护学习进度、错题本、薄弱点、通关记录和复习调度
- 支持暂停后从检查点继续，无需重新开始整个复习流程
- 根据答题表现动态调整后续路线，而不是机械照着目录推进

## 与普通复习提纲的区别

普通提示词通常一次性输出完整总结，内容生成后就结束了。FinaleForge Coach
维护的是一个持续学习过程：

```text
扫描资料
  → 确认资料与重点
  → 选择训练强度
  → 讲解一个小考点
  → 出题并等待回答
  → 批改与诊断
  → 记录错题和进度
  → 安排复习或进入下一考点
```

它会保存当前状态，知道哪些内容尚未学习、哪些内容需要复习、哪些考点已经达到
通关条件。

## 适合在这些情况下使用

- “根据这个文件夹里的资料带我复习期末考试”
- “老师划了重点，请先整理考点，再一个个教我”
- “不要一次讲完，讲一点就考我一道题”
- “根据真题和样卷判断哪些内容更重要”
- “从上次的学习进度继续”
- “集中复习我的错题和薄弱知识点”
- “重新扫描资料，并更新我的复习路线”

## 安装

将仓库克隆到 Codex Skills 目录：

```powershell
git clone https://github.com/hpsyyds-sys/finale-forge-coach.git `
  "$HOME\.codex\skills\finale-forge-coach"
```

也可以下载仓库中的 `finale-forge-coach.zip`，解压后把
`finale-forge-coach` 文件夹放入：

```text
~/.codex/skills/
```

安装后的关键结构：

```text
finale-forge-coach/
├── SKILL.md
├── agents/
│   └── openai.yaml
├── references/
└── templates/
```

## 使用示例

把课程资料放在同一个工作目录中，然后对 Codex 说：

```text
使用 $finale-forge-coach，扫描当前文件夹中的资料，先建立考点索引，
让我确认学习路线后，再一次带我学习一个小考点。
```

继续上次学习：

```text
使用 $finale-forge-coach，从上次保存的进度继续。
```

错题专项：

```text
使用 $finale-forge-coach，根据错题本带我进行薄弱点复习。
```

## 核心原则

- 老师划重点、考试范围、真题和样卷拥有更高优先级。
- 正式学习前必须完成资料索引、用户确认和训练强度选择。
- 每次只处理一个足够小的考点。
- 出题后必须等待用户回答，不能立即给出答案。
- 错误和不完整回答必须进入错题与复习调度。
- 通关由训练强度、答题表现和连续正确情况共同决定。
- 除非用户明确暂停或退出，否则保持交互式学习流程。

## 仓库内容

- `SKILL.md`：Skill 入口、触发描述和核心执行流程
- `agents/openai.yaml`：Codex 界面名称、简介和默认提示词
- `references/`：状态机、教学、文件处理和复习调度规则
- `templates/`：资料盘点、学习路线、进度表、错题本等输出模板
- `finale-forge-coach.zip`：可直接下载并安装的打包版本
